
## **Module 1 | Lesson 2: Xcode Essentials: Canvas, Previews, and the Inspector**

ၼႂ်း Lesson ၼႆႉ ႁဝ်းတေမႃးႁဵၼ်းႁူႉလွင်ႈ "ၶိူင်ႈၵမ်ႉၵႅမ်" ၼႂ်း Xcode ဢၼ်တေၸွႆႈႁႂ်ႈႁဝ်းတႅမ်ႈ UI လႆႈၽႂ်းၼၼ်ႉၶႃႈ။



### **1. The Canvas (တီႈၼႄႁၢင်ႈၽၢင်)**
**Canvas** ၼႆႉ ပဵၼ်ၼႃႈလိၵ်ႈၽၢႆႇၶႂႃ ဢၼ်ၼႄႁၢင်ႈ SwiftUI View ႁဝ်းၼၼ်ႉၶႃႈ။
*   မိူဝ်ႈႁဝ်းလႅၵ်ႈလၢႆႈ Code ၽၢႆႇသၢႆႉၼၼ်ႉ၊ ၼႃႈၸေႃး ၽၢႆႇၶႂႃ (Canvas) တေလႅၵ်ႈလၢႆႈၸွမ်းၵမ်းလဵဝ်ယဝ်ႉ။
*   ႁဝ်းဢမ်ႇလူဝ်ႇ Build & Run တီႈ Simulator တႃႇသေႇ၊ ႁဵတ်းႁႂ်ႈႁဝ်းလူတ်းယွမ်းၶၢဝ်းယၢမ်း (Save time) လႆႈၼမ်ၼႃႇ။

### **2. Previews (`#Preview`)**
ၼႂ်း SwiftUI ၼႆႉ ႁဝ်းၸႂ်ႉ Macro ဢၼ်ၸိုဝ်ႈဝႃႈ `#Preview` တႃႇသင်ႇႁႂ်ႈ Canvas ၼႄႁၢင်ႈယဝ်ႉ။
*   ႁဝ်းၵုမ်းထိင်းလႆႈဝႃႈ ၶႂ်ႈႁႂ်ႈမၼ်းၼႄႁၢင်ႈၼိူဝ် Device မဵဝ်းလႂ် (iPhone 15, iPad, ဢမ်ႇၼၼ် Mac)။
*   **Pro Tip:** ႁဝ်းၸၢင်ႈသၢင်ႈ Preview လၢႆလၢႆဢၼ် (Multiple Previews) ၼႂ်း File လဵဝ်ၵၼ်၊ မိူၼ်ၼင်ႇ ဢၼ်ၼိုင်ႈၼႄ **Light Mode** လႄႈ ထႅင်ႈဢၼ်ၼိုင်ႈၼႄ **Dark Mode** ၼၼ်ႉၶႃႈ။

### **3. The Inspector (တီႈၵုမ်းထိင်း Property)**
**Inspector** ပဵၼ် Panel ၽၢႆႇၶႂႃသုတ်းသုတ်း။ မၼ်းမိူၼ်ၼင်ႇ "Property Window" တီႈ Android Studio ၼၼ်ႉၶႃႈ။
*   ႁဝ်းၸၢင်ႈလႅၵ်ႈလၢႆႈ Font, Color, Alignment လႄႈ Padding လႆႈလူၺ်ႈလၢႆး "Click" ၵူၺ်း။
*   မိူဝ်ႈႁဝ်း Click လႅၵ်ႈလၢႆႈတီႈ Inspector ယဝ်ႉ၊ Xcode တေတႅမ်ႈ **Modifier Code** သႂ်ႇပၼ်ၼႂ်း Editor ႁဝ်းႁင်းမၼ်း (Auto-generate code)။

### **4. Live vs. Selectable Mode**
ၼႂ်း Canvas ၼၼ်ႉ မၼ်းမီးလၢႆးလဵၼ်ႈ 2 မဵဝ်း:
*   **Selectable (Selection Mode):** မိူဝ်ႈႁဝ်း Click တီႈ UI Element ၼၼ်ႉ၊ မၼ်းတေ Highlight Code တီႈ Editor ႁႂ်ႈႁဝ်းႁူႉဝႃႈ မၼ်းပဵၼ် Code တီႈလႂ်။
*   **Live Mode:** ႁဝ်းၸၢင်ႈလဵၼ်ႈၸွမ်း UI (Interactivity) လႆႈၵမ်းလဵဝ် မိူၼ်ၼင်ႇပဵၼ်ၼႂ်း App တေႉတေႉၼၼ်ႉၶႃႈ (Tap buttons, Scroll list, etc.)။

---

### **Summary (သူးပ်းႁူဝ်တွၼ်ႈႁဵၼ်း)**

| Component | Explanation (Shan) |
| :--- | :--- |
| **Canvas** | ပဵၼ်တီႈၼႄႁၢင်ႈ UI ႁဝ်းၵမ်းလဵဝ် (Real-time Preview)။ |
| **#Preview** | ပဵၼ် Code တႃႇသင်ႇႁႂ်ႈ Xcode ၼႄႁၢင်ႈ View တီႈ Canvas။ |
| **Inspector** | ၶိူင်ႈမိုဝ်းတႃႇလႅၵ်ႈလၢႆႈ Property (Fonts, Colors) လူၺ်ႈလၢႆး Click ၵူၺ်း။ |
| **Modifier Auto-gen** | မိူဝ်ႈလႅၵ်ႈလၢႆႈတီႈ Inspector၊ Xcode တေတႅမ်ႈ Code သႂ်ႇပၼ်ႁင်းမၼ်း။ |
| **Live Mode** | လၢႆးၸၢမ်းလဵၼ်ႈ UI (Button click/Scroll) ၼႂ်း Preview ၵမ်းလဵဝ်။ |
