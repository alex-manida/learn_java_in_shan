## Lesson 62: Cleanup with defer: Ensuring Code Runs Regardless of Errors

ၵမ်းမၢင် ႁဝ်းမီးၵၢၼ်ဢၼ်တေလႆႈ "ပိတ်း" (Close) ဢမ်ႇၼၼ် "လႅၵ်ႈၶိုၼ်း" သေႇသေႇ ဝၢႆးလင်ႁဵတ်းၵၢၼ်ယဝ်ႉ (မိူၼ်ၼင်ႇ ပိတ်း File, ပိတ်း Database Connection, ဢမ်ႇၼၼ် ပိတ်း Loading Spinner)။

သင်ဝႃႈ Code ႁဝ်းမီး `return` လၢႆလၢႆတီႈ ဢမ်ႇၼၼ်မီး `throw error`၊ ႁဝ်းၸၢင်ႈလိုမ်းတႅမ်ႈ Code ပိတ်းၸိူဝ်းၼၼ်ႉၶႃႈ။ Swift ၸင်ႇမီး **`defer`** တွၼ်ႈတႃႇၸွႆးႁဝ်းၶႃႈ။

### 1. `defer` ႁဵတ်းၵၢၼ်ၸိူင်ႉၼင်ႇႁိုဝ်?
Code ဢၼ်ယူႇၼႂ်း **`defer`** block တေပႂ်ႉတေႃႇပေႃး Function ၼၼ်ႉ "တေၸတ်းၵႂႃႇ" (Exit) ၸင်ႇတေႁဵတ်းၵၢၼ်ၶႃႈ။ ဢမ်ႇဝႃႈမၼ်းတေၸတ်းၵႂႃႇလူၺ်ႈ `return` ထမ်ႇမတႃႇ ဢမ်ႇၼၼ်လူၺ်ႈ `throw error` ၵေႃႈယႃႇ၊ `defer` တေႁဵတ်းၵၢၼ်သေႇသေႇၶႃႈ။

```swift
func processFile() {
    print("1. ပိုတ်ႇ File ၶႃႈ")
    
    defer {
        print("4. ပိတ်း File သေႇသေႇ (Cleanup)")
    }
    
    print("2. ဢၢၼ်ႇၶေႃႈมูล...")
    
    let hasError = true
    if hasError {
        print("3. မီးပၼ်ႁႃ! တေဢွၵ်ႇယဝ်ႉ")
        return // ၵမ်းၼႆႉ defer တေႁဵတ်းၵၢၼ်ၵမ်းလဵဝ်
    }
    
    print("ၵၢၼ်ဢၼ်တေဢမ်ႇႁွတ်ႈထိုင်...")
}
```

---

### 2. ပၵ်းပဝ်းၵၢၼ်ၸႂ်ႉ `defer`
1.  **Last In, First Out (LIFO):** သင်ႁဝ်းမီး `defer` လၢႆလၢႆဢၼ်၊ တူဝ်ဢၼ်ယူႇ **ၽၢႆႇတႂ်ႈသုတ်း** တေႁဵတ်းၵၢၼ် **ဢွၼ်တၢင်းပိူၼ်ႈ** ၶႃႈ။
2.  **Scope Based:** မၼ်းတေႁဵတ်းၵၢၼ် မိူဝ်ႈမၼ်းဢွၵ်ႇၼႂ်း `{ }` ဢၼ်မၼ်းယူႇဝႆႉၼၼ်ႉၶႃႈ။
3.  **No Return:** ၼႂ်း `defer` block ႁဝ်းဢမ်ႇၸၢင်ႈၸႂ်ႉ `return` ဢမ်ႇၼၼ် `throw` ဢွၵ်ႇမႃးလႆႈၶႃႈ။

---

### 3. ၵွပ်ႈသင်လူဝ်ႇၸႂ်ႉ defer?
1.  **Reliability:** မၼ်ႈၸႂ်လႆႈဝႃႈ ၵၢၼ် Cleanup တေထုၵ်ႇႁဵတ်းသေႇသေႇ။
2.  **Clarity:** ႁဝ်းၸၢင်ႈတႅမ်ႈ Code "ပိတ်း" ဝႆႉၸပ်းၵၼ်တင်း Code "ပိုတ်ႇ" တႄႇတႄႉမၼ်း ႁႂ်ႈဢၢၼ်ႇလႆႈငၢႆႈ။
3.  **Prevents Leaks:** ႁႄႉၵင်ႈပၼ်ႁႃ Memory ဢမ်ႇၼၼ် File ဢၼ်ပိုတ်ႇၶင်ဝႆႉၼၼ်ႉယဝ်ႉ။
