## Lesson 10: Arrays (Part 2): Advanced Methods (Map, Filter, Reduce)

ၼႂ်း Swift ၼႆႉ ႁဝ်းဢမ်ႇလူဝ်ႇတႅမ်ႈ Loop ယၢဝ်းယၢဝ်းတွၼ်ႈတႃႇလႅၵ်ႈလၢႆႈၶေႃႈမုၼ်း ၵူႈပွၵ်ႈၵေႃႈလႆႈယဝ်ႉ။ 
Swift မီး **Higher-Order Functions** ဢၼ်တေၸွႆးႁႂ်ႈႁဝ်းတႅမ်ႈ Code ဢၼ် "သႅၼ်ႈသႂ်" (Clean Code) လႄႈ ႁဵတ်းၵၢၼ်လႆႈ ၽႂ်းယဝ်ႉ။

### 1. Map: ၵၢၼ်လႅၵ်ႈလၢႆႈၶေႃႈမုၼ်း (Transforming)
`map` တေဢဝ်ၶေႃႈမုၼ်း ၵူႈတူဝ်ၼႂ်း Array မႃးလႅၵ်ႈပဵၼ်မဵဝ်းမႂ်ႇ ၸွမ်းၼင်ႇႁဝ်းသင်ႇဝႆႉယဝ်ႉ။ 
ဝၢႆးၼၼ်ႉယဝ်ႉ မၼ်းတေ ထုတ်ႇဢွၵ်ႇမႃးပဵၼ် Array မႂ်ႇ ဢၼ်မီးတူဝ်ၼမ်မိူၼ်ၵဝ်ႇယဝ်ႉ။

> **တူဝ်ယၢင်ႇ:** သင်ဝႃႈႁဝ်းမီး Array ၶေႃႈၵႂၢမ်းဢင်းၵိတ်း သေၶႂ်ႈလႅၵ်ႈပဵၼ် တူဝ်ယႂ်ႇတင်းသဵင်ႈ။

```swift
let keys = ["a", "b", "c"]
let upperKeys = keys.map { $0.uppercased() } 
// Output: ["A", "B", "C"]
```
*   `$0` မၢႆထိုင် တူဝ်လိၵ်ႈၼိုင်ႈတူဝ် ဢၼ်တိုၵ်ႉထုၵ်ႇၸတ်းၵၢၼ်ယူႇၼၼ်ႉယဝ်ႉ။

---

### 2. Filter: ၵၢၼ်လိူၵ်ႈဢဝ်ၶေႃႈမုၼ်း (Selecting)
`filter` တေၵူတ်ႇထတ်းၶေႃႈမုၼ်း ၵူႈတူဝ်ယဝ်ႉ။ တူဝ်လႂ်ဢၼ် "မၢၼ်ႇမႅၼ်ႈ" (True) ၸွမ်းၼင်ႇသၢႆငၢႆ (Condition) ႁဝ်းၼၼ်ႉ မၼ်းတေၵဵပ်းဝႆႉသေ တူဝ်ဢၼ်ဢမ်ႇမၢၼ်ႇၼၼ်ႉ မၼ်းတေထိုမ်ႈပႅတ်ႈယဝ်ႉ။

> **တူဝ်ယၢင်ႇ:** သင်ဝႃႈႁဝ်းၶႂ်ႈလိူၵ်ႈဢဝ် "မၢႆသဵင်" (Tone Marks) ၼႂ်းသဵၼ်ႈမၢႆတူဝ်လိၵ်းတႆး။

```swift
let allCharacters = ["ၵ", "ၶ", "်", "င", "း"]
let toneMarks = allCharacters.filter { $0 == "်" || $0 == "း" }
// Output: ["်", "း"]
```

---

### 3. Reduce: ၵၢၼ်ႁူမ်ႈၶေႃႈမုၼ်း (Combining)
`reduce` တေဢဝ်ၶေႃႈမုၼ်း တင်းသဵင်ႈၼႂ်း Array မႃး "ႁူမ်ႈၵၼ်" ႁႂ်ႈၵိုတ်းပဵၼ်ၵႃႈၶၼ် (Value) တူဝ်လဵဝ်ၵူၺ်းယဝ်ႉ။

> **တူဝ်ယၢင်ႇ:** ဢဝ်တူဝ်လိၵ်ႈလၢႆလၢႆတူဝ် မႃးႁူမ်ႈၵၼ်ပဵၼ် ၶေႃႈၵႂၢမ်း (String) ၼိုင်ႈၶေႃႈ။

```swift
let rowLetters = ["တ", "ႆ", "း"]
let fullWord = rowLetters.reduce("") { $0 + $1 }
// Output: "တႆး"
```
*   `""` တီႈၼႃႈသုတ်းၼၼ်ႉ ပဵၼ် "ၵႃႈၶၼ်တႄႇတႄႇ" (Initial Value)။
*   `$0` ပဵၼ်ၵႃႈၶၼ်ဢၼ်ႁူမ်ႈမႃးဝႆႉယဝ်ႉ (Result)၊ `$1` ပဵၼ်တူဝ်မႂ်ႇဢၼ်တိုၵ်ႉမႃးႁူမ်ႈ (Next item)။

---

### 4. Sort: ၵၢၼ်ၸတ်းထႅဝ် (Sorting)
*   **`.sorted()`**: ၸတ်းထႅဝ်ပၼ်မႂ်ႇ (ဢွၵ်ႇမႃးပဵၼ် Array ဢၼ်မႂ်ႇ)။
*   **`.sort()`**: ၸတ်းထႅဝ်တီႈ Array တူဝ်ၵဝ်ႇ (လူဝ်ႇပဵၼ် `var` ၵူၺ်း ၸင်ႇၸႂ်ႉလႆႈ)။

```swift
var scores = [80, 100, 90]
let sortedScores = scores.sorted() // [80, 90, 100]
```
