## Lesson 21: While and Repeat-While Loops

ပေႃးပဵၼ် `for-in` ႁဝ်းႁူႉဝႆႉယဝ်ႉဝႃႈ တေႁဵတ်းၵၢၼ်လၢႆပွၵ်ႈ။ 
ၵူၺ်းၵႃႈ **`while`** loop ၼႆႉ ႁဝ်းတေၸႂ်ႉမိူဝ်ႈႁဝ်း **ဢမ်ႇပႆႇႁူႉဝႃႈတေလႆႈႁဵတ်းလၢႆပွၵ်ႈ** သေ
မၼ်းတေသိုပ်ႇႁဵတ်းၵႂႃႇ တေႃႇမိူဝ်ႈသၢႆငၢႆပဵၼ် `false` ယဝ်ႉ။

### 1. While Loop (ၵူတ်ႇထတ်းဢွၼ်တၢင်း)
`while` တေၵူတ်ႇထတ်းသၢႆငၢႆ (Condition) တီႈႁူဝ်တႄႇမၼ်းဢွၼ်တၢင်းယဝ်ႉ။
သင်ဝႃႈသၢႆငၢႆပဵၼ် `false` တင်ႈတႄႇမိူဝ်ႈတႄႇမၼ်းၼႆ Code ၽၢႆႇၼႂ်းတေဢမ်ႇႁဵတ်းၵၢၼ်သေပွၵ်ႈၶႃႈ။

```swift
var count = 5

while count > 0 {
    print("Countdown: \(count)")
    count -= 1 // ယွမ်းၵႃႈၶၼ်လူင်း တေႃႇမိူဝ်ႈပဵၼ် 0
}
```

---

### 2. Repeat-While Loop (ႁဵတ်းဢွၼ်တၢင်း ၸင်ႇၵူတ်ႇထတ်း)
ၼႂ်းလိၵ်ႈတၢင်ႇပိူင် ႁဝ်းႁွင်ႉဝႃႈ `do-while` ယဝ်ႉ။ မၼ်းတေ **ႁဵတ်းၵၢၼ်ၼႂ်း Loop ဢွၼ်တၢင်း 1 ပွၵ်ႈ** ဢမ်ႇဝႃႈသၢႆငၢႆတေပဵၼ် true ႁႃႉ false ၵေႃႈယႃႇ၊ 
ယဝ်ႉၸင်ႇၵွႆၵႂႃႇၵူတ်ႇထတ်းသၢႆငၢႆတီႈၽၢႆႇတႂ်ႈမၼ်းၶႃႈ။

```swift
var i = 1

repeat {
    print("ႁဵတ်းၵၢၼ်ပွၵ်ႈၵမ်း: \(i)")
    i += 1
} while i <= 3
```



---

### 3. လွင်ႈပႅၵ်ႇပိူင်ႈၵၼ်
*   **`while`**: တူၺ်းသၢႆငၢႆဢွၼ်တၢင်း -> ၸင်ႇႁဵတ်းၵၢၼ်။
*   **`repeat-while`**: ႁဵတ်းၵၢၼ်ဢွၼ်တၢင်း -> ၸင်ႇတူၺ်းသၢႆငၢႆ။

### 4. Infinite Loops (Loop ဢၼ်ဢမ်ႇမီးဝၼ်းသုတ်း)
သင်ဝႃႈႁဝ်းလိုမ်း "လႅၵ်ႈလၢႆႈၵႃႈၶၼ်" ၼႂ်းသၢႆငၢႆ (မိူၼ်ၼင်ႇလိုမ်းတႅမ်ႈ `count -= 1`) ၼႆ 
Loop ၼၼ်ႉတေႁဵတ်းၵၢၼ်ၵႂႃႇတေႃႇၸူဝ်ႈ (Infinite Loop) သေ ႁဵတ်းႁႂ်ႈ App ႁဝ်းၵိုတ်း (Hang) ၵႂႃႇလႆႈၶႃႈ။
