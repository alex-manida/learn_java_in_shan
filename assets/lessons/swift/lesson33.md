## Lesson 33: The 'Optional' Concept: Why Swift Doesn't Allow Nulls

ၼႂ်းလိၵ်ႈ Programming တၢင်ႇမဵဝ်း (မိူၼ်ၼင်ႇ Java ဢမ်ႇၼၼ် C#) ၼၼ်ႉ Variable ၸၢင်ႈပဵၼ် **`null`** (ပဝ်ႇဝႆႉ/ဢမ်ႇမီးၶေႃႈမုၼ်း) လႆႈၵူႈတူဝ်ယဝ်ႉ။
ပေႃးႁဝ်းလိုမ်းၵူတ်ႇထတ်းသေ ၵႂႃႇၸႂ်ႉ Variable ဢၼ်ပဵၼ် `null` ၼၼ်ႉ App တေ Crash ၵမ်းလဵဝ်ယဝ်ႉ။
ဢၼ်ၼႆႉပဵၼ်ပၼ်ႁႃလူင် ဢၼ်ၵူၼ်းတႅမ်ႈ Code ၵူဝ်သုတ်းယဝ်ႉ။

Swift ၸင်ႇၵႄႈပၼ်ႁႃၼႆႉလူၺ်ႈ **Optionals** ယဝ်ႉ။

### 1. Swift ဢမ်ႇပၼ်မီး Null (By Default)
ၼႂ်း Swift, Variable ထမ်ႇမတႃႇ **တေလႆႈမီးၵႃႈၶၼ်တႃႇသေႇ** ယဝ်ႉ။ မၼ်းဢမ်ႇပၼ်ႁဝ်းသႂ်ႇ `nil` (ၶေႃႈၵႂၢမ်း Swift ဢၼ်ၸႂ်ႉတႅၼ်း null) ၶဝ်ႈၵႂႃႇၼႂ်း Variable ဢၼ်ဢမ်ႇၸႂ်ႉ Optional ယဝ်ႉ။

```swift
var name: String = "Sai Mao"
// name = nil // Error: String ဢမ်ႇပၼ်ပဵၼ် nil လႆႈ
```

### 2. Optional မၢႆထိုင်သင်?
**Optional** မိူၼ်ၼင်ႇ "ၵွၵ်း" (Box) ဢၼ်ႁဝ်းႁေႃႇဝႆႉယဝ်ႉ။
*   ၼႂ်းၵွၵ်းၼၼ်ႉ ၸၢင်ႈမီး **"ၶေႃႈမုၼ်း"** ယူႇ။
*   ဢမ်ႇၼၼ် ၼႂ်းၵွၵ်းၼၼ်ႉ ၸၢင်ႈ **"ပဝ်ႇဝႆႉ"** (`nil`) ယဝ်ႉ။

ႁဝ်းၸႂ်ႉမၢႆ **`?`** တွၼ်ႈတႃႇမၵ်းမၼ်ႈဝႃႈ Variable ၼႆႉပဵၼ် Optional ယဝ်ႉ။



```swift
var nickname: String? = "Mao" // မီးၶေႃႈမုၼ်း "Mao" ယူႇၼႂ်းၵွၵ်း
nickname = nil                // ယၢမ်းလဵဝ် ၵွၵ်းပဝ်ႇဝႆႉယဝ်ႉ
```

---

### 3. ၵွပ်ႈသင်လၢႆးၼႆႉသမ်ႉလီ?
1.  **Safety (လွင်ႈလွတ်ႈၽေး):** Swift တေႁႂ်ႈႁဝ်း "ၵူတ်ႇထတ်း" (Unwrap) ၵွၵ်းၼၼ်ႉဢွၼ်တၢင်း ၸင်ႇတေပၼ်ၸႂ်ႉ ၶေႃႈမုၼ်းၽၢႆႇၼႂ်းယဝ်ႉ။ မၼ်းႁဵတ်းႁႂ်ႈႁဝ်းဢမ်ႇလိုမ်းၵူတ်ႇထတ်း `nil` ယဝ်ႉ။
2.  **Clarity (လွင်ႈၸႅင်ႈလႅင်း):** ပေႃးႁဝ်းလူ Code ၼႆ ႁဝ်းတေႁူႉလႆႈၵမ်းလဵဝ်ဝႃႈ Variable တူဝ်လႂ် ၸၢင်ႈပဵၼ် `nil` လႆႈ လႄႈ တူဝ်လႂ် ဢၼ်မီးၶေႃႈမုၼ်းသေႇသေႇ (Always has a value) ၼႆယဝ်ႉ။
