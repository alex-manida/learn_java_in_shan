## Lesson 23: Labeled Statements: Managing Complex Nested Loops

မိူဝ်ႈႁဝ်းတႅမ်ႈ Code ဢၼ်မီး Loop သႂ်ႇၼႂ်း Loop (Nested Loops) ၼၼ်ႉ ၵမ်းမၢင်ႁဝ်းၸၢင်ႈယုင်ႈယၢင်ႈဝႃႈ "ပေႃးႁဝ်းတႅမ်ႈ `break` တီႈၼႆႉ မၼ်းတေၵိုတ်းတူဝ်လႂ်?" ၼႆယဝ်ႉ။ **Labeled Statements** တေၸွႆးႁႂ်ႈႁဝ်းၸီႉသင်ႇလႆႈ တိူဝ်းတႅတ်ႈတေႃးလိူဝ်ၵဝ်ႇၶႃႈ။

### 1. Syntax ၵၢၼ်တင်ႈၸိုဝ်ႈ
ႁဝ်းၸၢင်ႈဢဝ်ၸိုဝ်ႈ (Name) သႂ်ႇဝႆႉၽၢႆႇၼႃႈ `for`, `while`, ဢမ်ႇၼၼ် `switch` သေ ၸွမ်းလူၺ်ႈမၢႆ `:` (Colon) ၶႃႈ။

```swift
outerLoop: for i in 1...3 {
    innerLoop: for j in 1...3 {
        if i == 2 && j == 2 {
            break outerLoop // သင်ႇႁႂ်ႈၵိုတ်းတင်း Loop ၽၢႆႇၼွၵ်ႈၵမ်းလဵဝ်
        }
        print("i: \(i), j: \(j)")
    }
}
```

---

### 2. ၵၢၼ်ၸႂ်ႉ `continue` ၸွမ်း Label
ႁဝ်းၵေႃႈၸၢင်ႈၸႂ်ႉ `continue` ၸွမ်းၸိုဝ်ႈ Label လႆႈမိူၼ်ၵၼ်ၶႃႈ။ မၼ်းတေၸွႆးႁႂ်ႈႁဝ်း "ၶၢမ်ႈ" ၵႂႃႇတႄႇတူဝ်မႂ်ႇ တီႈ Loop တူဝ်ဢၼ်ႁဝ်းၶႂ်ႈလႆႈတႄႇတႄႇၼၼ်ႉယဝ်ႉ။



---

### 3. ၵွပ်ႈသင်လူဝ်ႇၸႂ်ႉ?
1.  **Precision:** ၸီႉသင်ႇလႆႈတႅတ်ႈတေႃး (ၵိုတ်းတူဝ်ၼွၵ်ႈ ဢမ်ႇၼၼ် တူဝ်ၼႂ်း)။
2.  **Readability:** လူငၢႆႈလိူဝ်ၵဝ်ႇ ၵွပ်ႈႁဝ်းႁူႉဝႃႈ Loop ၼိုင်ႈၸုမ်းၼၼ်ႉ မၼ်းႁဵတ်းၵၢၼ်သင် (မိူၼ်ၼင်ႇ `rowLoop` လႄႈ `columnLoop`)။
