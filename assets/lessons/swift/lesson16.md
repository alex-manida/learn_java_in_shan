## Lesson 16: Memory Management of Collections: Value Semantics and Copy-on-Write (COW)

ၼႂ်းလိၵ်ႈ Swift ၼႆႉ Array, Dictionary, ลႄႈ Set တင်းသဵင်ႈၼၼ်ႉ ပဵၼ် **Value Types**။ ဢၼ်ၼႆႉပဵၼ်လွင်ႈလွတ်ႈၽေး (Safety) ဢၼ်ၶိုၵ်ႉတွၼ်းပိူၼ်ႈ၊ ၵူၺ်းၵႃႈ Swift ယັງမီးလၢႆးႁဵတ်းၵၢၼ်လႅတ်းၽၢႆ တွၼ်ႈတႃႇႁၵ်ႉသႃ Performance ႁဝ်းထႅင်ႈယူႇၶႃႈ။

### ### 1. Value Semantics (ၵၢၼ်ၵေႃႇပီႇၶေႃႈมูล)
မိူဝ်ႈႁဝ်းဢဝ် Array ဢၼ်ၼိုင်ႈ သႂ်ႇၼႂ်း Variable ထႅင်ႈတူဝ်ၼိုင်ႈ၊ Swift တေႁဵတ်းႁႂ်ႈမၼ်းပဵၼ် "ၶေႃႈมูลဢၼ်ဢမ်ႇၵဵဝ်ႇၶေႃၵၼ်"။ ပေႃးႁဝ်းလႅၵ်ႈတူဝ်ၼိုင်ႈ တူဝ်ၼိုင်ႈသမ်ႉတေဢမ်ႇလႅၵ်ႈၸွမ်းၶႃႈ။

```swift
var row1 = ["ၵ", "ၶ", "င"]
var row2 = row1 // ၵေႃႇပီႇ (Copy) ၵႂႃႇ

row2.append("ၸ")

print(row1) // ["ၵ", "ၶ", "င"] (မိူၼ်ၵဝ်ႇ)
print(row2) // ["ၵ", "ၶ", "င", "ၸ"] (လႅၵ်ႈလၢႆႈၵႂႃႇ)
```

### 2. Copy-on-Write (COW)
ၸၢႆးမၢဝ်း ၸၢင်ႈဝူၼ်ႉဝႃႈ "ပေႃးႁဝ်းမီး Array ဢၼ်ယႂ်ႇၼႃႇ (မိူၼ်ၼင်ႇ Dictionary တူဝ်လိၵ်ႈ 10,000 တူဝ်) သေ ၵေႃႇပီႇၵူႈပွၵ်ႈၼႆ Memory တေဢမ်ႇတဵမ်ႁႃႉ?"

Swift မီးၼမ်ႉၵတ်ႉ **Copy-on-Write**:
*   မိူဝ်ႈႁဝ်းတႅမ်ႈ `row2 = row1` ၼၼ်ႉ Swift တေ**ပႆႇၵေႃႇပီႇတႄႇတႄႇ**။ မၼ်းတေပၼ်တင်းသွင်တူဝ် ၸႂ်ႉ Memory တီႈလဵဝ်ၵၼ်ဝႆႉဢွၼ်တၢင်း (တွၼ်ႈတႃႇႁႂ်ႈမၼ်းၽႂ်း)။
*   တေႃႇမိူဝ်ႈ `row2` တႄႇမီး**ၵၢၼ်လႅၵ်ႈလၢႆႈ** (Write) ၼၼ်ႉၵူၺ်း Swift ၸင်ႇတေၵေႃႇပီႇၶေႃႈมูลဢွၵ်ႇမႃးပဵၼ်တူဝ်မႂ်ႇတႄႇတႄႇယဝ်ႉ။

### 3. ၵွပ်ႈသင်လူဝ်ႇႁူႉလွင်ႈၼႆႉ?
တွၼ်ႈတႃႇ Developer ၼၼ်ႉ မၼ်းၸွႆးႁႂ်ႈႁဝ်း:
1.  **Thread Safety:** ႁဝ်းသူင်ႇ Array ၵႂႃႇၸႂ်ႉၼႂ်း Thread တၢင်ႇဢၼ်လႆႈငၢႆႈငၢႆႈ ၵွပ်ႈႁဝ်းႁူႉဝႃႈ မၼ်းတေဢမ်ႇၵႂႃႇလႅၵ်ႈလၢႆႈၸူးတူဝ်တႄႇမၼ်း။
2.  **Performance:** ႁဝ်းဢမ်ႇလူဝ်ႇၵူဝ်လွင်ႈသူင်ႇ Array ၶဝ်ႈၼႂ်း Function ၵွပ်ႈ Swift တေၸတ်းၵၢၼ် Memory ႁႂ်ႈငၢႆႈလႄႈၽႂ်းပိူၼ်ႈယဝ်ႉ။
