

## **Lesson 45: Kotlin Coroutines (Background Tasks)**

### **1. The Main Thread Rule (ပၵ်းပိူင် "သဵၼ်ႈတၢင်းလူင်")**

ၼႂ်း Androi ၼႆႉ မီးသဵၼ်ႈတၢင်း (Thread) ၼိုင်ႈဢၼ် ဢၼ်လမ်ႇလွင်ႈသုတ်း ႁွင်ႉဝႃႈ **Main Thread (UI Thread)** ၼႆယဝ်ႉ။

* **ၼႃႈတီႈ:** မၼ်းမီးၼႃႈတီႈ ၼႄႁၢင်ႈ (Drawing UI) လႄႈ ႁပ်ႉၵၢၼ်တူင်ႉၼိုင် (Touch Events) ယဝ်ႉ။
* **ပၵ်းပိူင်:** Android ဢမ်ႇပၼ်ႁႂ်ႈႁဝ်းႁွင်ႉ API ၼႂ်း Main Thread။
* **ပဵၼ်သင်?:** ယွၼ်ႉၵၢၼ်ႁွင်ႉ API ၼႆႉ ၵမ်ႈပႃႈၼမ် လႆးပႂ်ႉႁိုင် (1-3 ၸႅတ်ႉၵႅၼ်ႉ) ယဝ်ႉ။ သင်ႁဝ်းႁဵတ်းၼႂ်း Main Thread ၼႆ App တေၵိုတ်းၶမ်ဝႆႉသေ ၵူၼ်းၸႂ်ႉတေတဵၵ်းသင်ၵေႃႈဢမ်ႇလႆႈယဝ်ႉ။ ၼႆႉၸၢင်ႊႁဵတ်းႁႂ်ႈ App ႁဝ်းလူႉ (Crash) လူၺ်ႈ Error ဢၼ်ႁွင်ႉဝႃႈ `NetworkOnMainThreadException` ၼႆယဝ်ႉ။

### **2. Suspend Functions (ၵၢၼ် "ၵိုတ်း" ပႂ်ႉလူၺ်ႈ ဢမ်ႇပၼ်ၶမ်)**

တွၼ်ႈတႃႇၵႄႈပၼ်ႁႃၼႆႉ Kotlin မီးဝႆႉ **`suspend`** function ၼႆယဝ်ႉ။

* **လွင်ႈႁဵတ်းၵၢၼ်:** မၼ်းပွင်ႇဝႃႈ "ၵိုတ်း (Suspend) ၵၢၼ်ႁဵတ်းၵၢၼ် တီႈသဵၼ်ႈတၢင်းၼႆႉဝႆႉၵွၼ်ႇသေ ၵႂႃႇႁဵတ်းၵၢၼ်ၼၵ်းၼၵ်း တီႈသဵၼ်ႈတၢင်းတၢင်ႇဢၼ်လႄႈ။ ပေႃးယဝ်ႉယဝ်ႉ ၸင်ႇၶိုၼ်းမႃးႁဵတ်းၶိုၼ်း တီႈၵဝ်ႇၼႃ" ၼႆယဝ်ႉ။
* **လွင်ႈလီ:** မၼ်းႁဵတ်းႁႂ်ႈ Code ႁဝ်းဢၢၼ်ႇငၢႆႈ မိူၼ်ၼင်ႇ Code ပၢၼ်ၵဝ်ႇ (Synchronous) ယဝ်ႉ။ ၵူၺ်းၵႃႈ မၼ်းႁဵတ်းၵၢၼ်ပဵၼ် Background ႁင်းၵွႆးမၼ်းယဝ်ႉ။

---

### **3. Dispatchers (ၵူၼ်းၸတ်ႉၵၢၼ် သဵၼ်ႈတၢင်း)**

**Dispatchers** ပဵၼ်ၵူၼ်း ဢၼ်လၢတ်ႈၼႄဝႃႈ တေႁႂ်ႈ Coroutine ၼၼ်ႉ လႅၼ်ႈၼိူဝ် "သဵၼ်ႈတၢင်း" လႂ်ၼႆယဝ်ႉ:

* **`Dispatchers.IO`:** ၸႂ်ႉတွၼ်ႈတႃႇၵၢၼ် Networking (Retrofit), ၵၢၼ်ပိုတ်ႇၾၢႆႇ, ဢမ်ႇၼၼ် ၵၢၼ်ၸႂ်ႉ Database (Room) ယဝ်ႉ။ မၼ်းထုၵ်ႇဢွၵ်ႇပိူင်မႃး တႃႇႁဵတ်းၵၢၼ် ဢၼ်ၸႂ်ႉၶၢဝ်းယၢမ်းႁိုင် မိူၼ်ၼင်ႇ Input/Output ယဝ်ႉ။
* **`Dispatchers.Main`:** ၸႂ်ႉမိူဝ်ႈႁဝ်းလႆႈၶေႃႈမုၼ်းမႃးယဝ်ႉသေ ၶႂ်ႈဢဝ်ၵႂႃႇၼႄၼိူဝ် UI (မိူၼ်ၼင်ႇ Update TextView) ယဝ်ႉ။
* **`Dispatchers.Default`:** ၸႂ်ႉတွၼ်ႈတႃႇၵၢၼ်ၼပ်ႉတူဝ်ၼပ်ႉ ဢၼ်ၼၵ်းၼၵ်း (CPU-intensive tasks) ယဝ်ႉ။

#### **တူဝ်ယၢင်ႇ Code ၼႂ်း ViewModel:**

```kotlin
fun fetchRecipes() {
    // ၸႂ်ႉ viewModelScope တႃႇတႄႇ Coroutine ဢၼ်တေတၢႆၸွမ်း ViewModel
    viewModelScope.launch(Dispatchers.IO) { 
        try {
            val result = repository.getRecipes() // ႁဵတ်းၵၢၼ်ၼႂ်း Background
            
            withContext(Dispatchers.Main) {
                // ဝၢႆးသေ လႆႈၶေႃႈမုၼ်းယဝ်ႉၼႆ မႃးၶိုၼ်းတီႈ Main Thread တႃႇ Update UI
                _recipes.value = result
            }
        } catch (e: Exception) {
            // ၸတ်ႉၵၢၼ် Error
        }
    }
}

```

### **Advice for Students**

'ဝူၼ်ႉတူၺ်းဝႃႈ Coroutine မိူၼ်ၼင်ႇ ၵူၼ်းၸွႆး (Helper) ၼႆၶႃႈလႄႈ။' 
ႁဝ်းမီးၵူၼ်းၸွႆးလၢႆလၢႆၵေႃႉ၊ ၵေႃႉၼိုင်ႈၵႂႃႇၵၢတ်ႇ (IO), ၵေႃႉၼိုင်ႈၵႂႃႇႁုင်ၶဝ်ႈ (Default), ယဝ်ႉၵေႃႉ ထႅင်ႈၵေႃႉၼိုင်ႈ (Main) သမ်ႉပႂ်ႉၼႄ "ၸိုဝ်ႈၶဝ်ႈၽၵ်း" ၼႆယဝ်ႉ။
သင်ႁဝ်းၸႂ်ႉၵူၼ်းၵေႃႉလဵဝ် ႁဵတ်းၵူႈပိူင်ၸိုင် ၶႅၵ်ႇတေလႆႈပႂ်ႉႁိုင် လႄႈ ၵမ်ႈၼမ် တေၸႂ်ဢမ်ႇယၢဝ်းသေ တေပၢႆႊဢွၵ်ႇၵႂႃႇၶိုၼ်းၶႃႈ။
