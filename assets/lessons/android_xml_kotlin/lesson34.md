
## **Lesson 34: The Problem with Traditional Code**

### **1. The "God Activity" (Activity ဢၼ်ႁဵတ်းၵၢၼ်ၵူႈပိူင်)**

မိူဝ်ႈႁဝ်းတႄႇႁဵၼ်းမႂ်ႇမႂ်ႇၼၼ်ႉ ႁဝ်းၵႆႉတႅမ်ႈ Code တင်းမူတ်းဝႆႉၼႂ်း `MainActivity.kt` ဢမ်ႇၼၼ် `Fragment` ယဝ်ႉ။

* **ပၼ်ႁႃ:** ၼႂ်း Activity ၼိုင်ႈဢၼ် သမ်ႉမီးတင်း ၵၢၼ်ၸတ်ႉၵၢၼ် UI, ၵၢၼ်တွပ်ႇတွၼ်ႈပုမ်ႇတဵၵ်း, ၵၢၼ်ၵူတ်ႇထတ်းၶေႃႈမုၼ်း, လႄႈ ၵၢၼ်ႁွင်ႉ API။
* **ၽွၼ်းလႆႈ:** ႁဵတ်းႁႂ်ႈ Code ႁဝ်းယၢဝ်းပူၼ်ႉတီႈ (ယၢပ်ႇတႃႇမႄး), ၵူတ်ႇထတ်း (Test) ယၢပ်ႇ, လႄႈ သင်မီးပၼ်ႁႃတီႈၼိုင်ႈ မၼ်းတေတုမ်ႉတိူဝ်ႉၵႂႃႇတင်းမူတ်းယဝ်ႉ။

---

### **2. Separation of Concerns (ၵၢၼ်ၸႅၵ်ႇၼႃႈတီႈ ၽႂ်မၼ်း)**

တွၼ်ႈတႃႇၵႄႈပၼ်ႁႃ "God Activity" ၼၼ်ႉ ႁဝ်းလူဝ်ႇၸႅၵ်ႇ "ၵၢၼ်ၼႄႁၢင်ႈ" (UI) ဢွၵ်ႇတီႈ "ပၢႆးဝူၼ်ႉ" (Logic) ယဝ်ႉ။

* **UI (View):** Activity/Fragment မီးၼႃႈတီႈ "ၼႄႁၢင်ႈ" လႄႈ "ႁပ်ႉၵၢၼ်တူင်ႉၼိုင်" ၵူၺ်း။ မၼ်းဢမ်ႇလူဝ်ႇႁူႉဝႃႈ ၶေႃႈမုၼ်းမႃးတီႈလႂ်။
* **Logic (ViewModel):** မီးၼႃႈတီႈ "ဝူၼ်ႉ" လႄႈ "ၵုမ်းထိင်းၶေႃႈမူၼ်း"။ မၼ်းတေဝၢင်းၽႅၼ်ဝႃႈ တေလူဝ်ႇၼႄသင်သေ ၸင်ႇသူင်ႇၵႂႃႇပၼ် UI ယဝ်ႉ။

---

### **3. Memory Leaks**

ပၼ်ႁႃလူင်ထႅင်ႈပိူင်ၼိုင်ႈၵေႃႈပဵၼ် မိူဝ်ႈႁဝ်းႁဵတ်းၵၢၼ် ဢၼ်ၸႂ်ႉၶၢဝ်းယၢမ်းႁိုင် (Long-running tasks) ၼႂ်း Activity ယဝ်ႉ။

* **ငဝ်းလၢႆး:** ၸဝ်ႈၵဝ်ႇသင်ႇႁႂ်ႈ Activity ၵႂႃႇ Download ၶႅမ်ႈႁၢင်ႈ (ႁိုင်မွၵ်ႈ 10 ၸႅတ်ႉၵႅၼ်ႉ)။ ၵူၺ်းၵႃႈ ၼႂ်းၵႄႈၵၢၼ်ၼၼ်ႉ ၵူၼ်းၸႂ်ႉ "ပိၼ်ႇၼႃႈၸေႃးၾူၼ်း" (Rotate Screen)။
* **ၽွၼ်းလႆႈ:** Android တေယႃႉ Activity တူဝ်ၵဝ်ႇပႅတ်ႈသေ သၢင်ႈတူဝ်မႂ်ႇ။ ၵူၺ်းၵႃႈ တူဝ် Download ၼၼ်ႉ သမ်ႉတိုၵ်ႉၵွင်ႉဝႆႉၸူး Activity တူဝ်ၵဝ်ႇ (ဢၼ်ထုၵ်ႇယႃႉၵႂႃႇယဝ်ႉၼၼ်ႉ)။
* **Result:** Android ဢမ်ႇၸၢင်ႈလၢင်ႉ Activity တူဝ်ၵဝ်ႇဢွၵ်ႇၼႂ်း RAM လႆႈ (Memory Leak) ယွၼ်ႉမၼ်းတိုၵ်ႉမီး ၵူၼ်းၵွင်ႉမၼ်းဝႆႉယူႇလႄႈ ၼႆႉႁဵတ်းႁႂ်ႈ App ၼၵ်း လႄႈ လူႉ (Crash) ယဝ်ႉ။

---

### **Advice for Students (ၶေႃႈသႅပ်ႇပၼ်လုၵ်ႈႁဵၼ်း):**

'Activity ပဵၼ်မိူၼ်ၼင်ႇ ၼႃႈတႃၽၢင်ႁၢင်ႈ ဢၼ်ၸၢင်ႈထုၵ်ႇပိၵ်ႉ လႄႈ ပိုတ်ႇမႂ်ႇလႆႈၵူႈယၢမ်းယဝ်ႉ။' 
ႁဝ်းဢမ်ႇထုၵ်ႇလီဢဝ်ၶေႃႈမုၼ်းလမ်ႇလွင်ႈ သႂ်ႇဝႆႉၼႂ်းမၼ်း။ ၵၢၼ်ၸႅၵ်ႇ Code ဢွၵ်ႇပဵၼ်တွၼ်ႈၽႂ်တွၼ်ႈမၼ်းၼႆႉ 
ဢမ်ႇၸႂ်ဝႃႈ ႁဵတ်းႁႂ်ႈႁဝ်းတႅမ်ႈ Code ၼမ်လိူဝ်ၵဝ်ႇလၢႆလၢႆၶႃႈ၊ ၵူၺ်းၵႃႈ မၼ်းႁဵတ်းႁႂ်ႈ App ႁဝ်း 'မၼ်ႈၵိုမ်း' လႄႈ 'မႄးငၢႆႈ' ၼႂ်းဝၼ်းၼႃႈၶႃႈ!
