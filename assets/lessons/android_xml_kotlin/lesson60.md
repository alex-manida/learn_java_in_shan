

## **Lesson 60: Performance & The Android Profiler**

### **1. The Layout Inspector (ၵၢၼ်ၵူတ်ႇထတ်း ၵၢၼ်ၼႄႁၢင်ႈ)**

မၢင်ပွၵ်ႈ App ႁဝ်းလႅၼ်ႈထိူင်း (Lag) မိူဝ်ႈႁဝ်းပၢၵ်ႈ (Scroll) ၼႃႈၸေႃးၼႆ ၼႆႉၵႆႉပဵၼ်ယွၼ်ႉ **Deep Layout Nesting** ယဝ်ႉ။

* **ပၼ်ႁႃ:** မိူၼ်ၼင်ႇႁဝ်းဢဝ် `LinearLayout` သႂ်ႇၼႂ်း `LinearLayout` သႂ်ႇၼႂ်း `RelativeLayout` သႂ်ႇၼႂ်း `FrameLayout` ထႅင်ႈလၢႆလၢႆၸၼ်ႉ။ Android လူဝ်ႇလႆႈၼပ်ႉတူဝ်ၼပ်ႉ (Calculate) သႅၼ်းမၼ်းလၢႆပွၵ်ႈ ႁဵတ်းႁႂ်ႈ CPU ႁဵတ်းၵၢၼ်ၼၵ်းယဝ်ႉ။
* **Layout Inspector:** ၶိူင်ႈၵူတ်ႇထတ်းၼႆႉ တေၼႄႁႂ်ႈႁဝ်းႁၼ်ဝႃႈ Layout ႁဝ်းမီးဝႆႉလၢႆၸၼ်ႉ။
* **လၢႆးၵႄႈ:** ၸႂ်ႉ **`ConstraintLayout`** တႃႇႁဵတ်းႁႂ်ႈ Layout ႁဝ်း "ပဵင်း" (Flat) သုတ်း တွၼ်ႈတႃႇလူတ်းယွမ်း ၵၢၼ်ႁဵတ်းၵၢၼ်ၶွင် CPU လႄႈ။

### **2. Memory Profiler (ၵၢၼ်ၵူတ်ႇထတ်း RAM)**

**Memory Leak** ပဵၼ်ပၼ်ႁႃဢၼ်ယႂ်ႇသုတ်း ဢၼ်ႁဵတ်းႁႂ်ႈ App လူႉ (Crash) သင်ၸႂ်ႉႁိုင်ၵႂႃႇဢိတ်းၼိုင်ႈယဝ်ႉ။

* **ပၼ်ႁႃ:** မၼ်းပဵၼ်ယွၼ်ႉငဝ်းလၢႆး ဢၼ် App ႁဝ်း "ယွၼ်းၸႂ်ႉ RAM" သေ ပေႃးဢမ်ႇၸႂ်ႉယဝ်ႉၵေႃႈ "ဢမ်ႇၶိုၼ်းပၼ်" (Not releasing) ၶိုၼ်းယဝ်ႉ။ ၼႆႉႁဵတ်းႁႂ်ႈ RAM ႁဝ်းတဵမ်သေ ၸင်ႇပဵၼ် Error `OutOfMemoryError` ၼႆယဝ်ႉ။
* **Memory Profiler:** မၼ်းတေၼႄသဵၼ်ႈၵၢင် (Graph) ၶွင် RAM ယဝ်ႉ။ သင်သဵၼ်ႈၵၢင်ၼၼ်ႉ ၶိုၼ်ႈသုင်ဝႆႉတႃႇသေႇလႄႈ ဢမ်ႇလူင်းမႃးသေပွၵ်ႈၼႆၸိုင် ပွင်ႇဝႃႈ App ၸဝ်ႈၵဝ်ႇမီး Memory Leak ယဝ်ႉ။

### **3. Energy Profiler (ၵၢၼ်ၵူတ်ႇထတ်း Battery)**

ၵူၼ်းၸႂ်ႉတေယႃႉ App ၸဝ်ႈၵဝ်ႇပႅတ်ႈၵမ်းလဵဝ် သင်ၶဝ်ႁၼ်ဝႃႈ App ၸဝ်ႈၵဝ်ႇ "ၵိၼ် Battery" ၼမ်ႁႅင်း ၼႆယဝ်ႉ။

* **ပၼ်ႁႃ:** ၵၢၼ်ႁွင်ႉ API တႃႇသေႇ၊ ၵၢၼ်ၸႂ်ႉ GPS တႃႇသေႇ၊ ဢမ်ႇၼၼ် ၵၢၼ်တႅမ်ႈၶေႃႈမၼ်းလူင်း Database ဢၼ်ဢမ်ႇလူဝ်ႇ။
* **Energy Profiler:** မၼ်းတေလၢတ်ႈၼႄႁဝ်းဝႃႈ မိူဝ်ႈလႂ် App ႁဝ်းၸႂ်ႉၾႆးၾႃႉၼမ်။ သင်ႁဝ်းႁၼ်ဝႃႈ မိူဝ်ႈႁဝ်းပိၵ်ႉ App ဝႆႉၵေႃႈ မၼ်းတိုၵ်ႉၵိၼ်ၾႆးယူႇၸိုင် ပွင်ႇဝႃႈႁဝ်းမီးၵၢၼ်ႁဵတ်းၵၢၼ်ၼႂ်း Background ဢၼ်ၽိတ်းပိူင်ဝႆႉၼႆယဝ်ႉ။
