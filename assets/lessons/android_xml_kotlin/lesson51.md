

## **Lesson 51: Defining the Entity (The Table)**

### **1. Data Mapping (ၵၢၼ်မၢႆတွင်းပဵၼ် Entity)**

တွၼ်ႈတႃႇလၢတ်ႈၼႄ Room ဝႃႈ Data Class တူဝ်ၼႆႉ ဢမ်ႇၸႂ် Class ထမ်ႇမထႃ၊ မၼ်းပဵၼ် "ပႅၼ်ႈသိမ်းၶေႃႈမူၼ်း" ၼႆၼၼ်ႉ ႁဝ်းတေလႆႈၸႂ်ႉ Annotation **`@Entity`** ဝႆႉၼိူဝ်ႁူဝ်မၼ်းယဝ်ႉ။

* **TableName:** ႁဝ်းၸၢင်ႈမၵ်းမၼ်ႈပၼ်ၸိုဝ်ႈ Table လႆႈ (သင်ဢမ်ႇသႂ်ႇ၊ မၼ်းတေဢဝ်ၸိုဝ်ႈ Class ပဵၼ်ၸိုဝ်ႈ Table ၵမ်းလဵဝ်ယဝ်ႉ)။

---

### **2. Primary Keys (သေႃးလပ်ႉပိူင်လူင်)**

ၼႂ်း Database ၼႆႉ ၶေႃႈမုၼ်းၵူႈထႅဝ် (Row) လူဝ်ႇလႆႈမီး "မၢႆတွင်း" ဢၼ်ဢမ်ႇမိူၼ်ပိူၼ်ႈ (Unique ID) တွၼ်ႈတႃႇႁႂ်ႈ Room ႁႃၶေႃႈမုၼ်းထူပ်းလႆႈငၢႆႈငၢႆႈယဝ်ႉ။

* **`@PrimaryKey`:** မၢႆတွင်းဝႃႈ Variable တူဝ်ၼႆႉ ပဵၼ် ID လူင်ၶွင် Row ၼႆႉယဝ်ႉ။
* **`autoGenerate = true`:** ၼႆႉလမ်ႇလွင်ႈၼႃႇၶႃႈ! မၼ်းပွင်ႇဝႃႈ ႁဝ်းဢမ်ႇလူဝ်ႇမႆႈၸႂ်လွင်ႈတႅမ်ႈမၢႆ ID ႁင်းၵွႆး။ ပေႃးႁဝ်းသႂ်ႇၶေႃႈမုၼ်းမႂ်ႇၶဝ်ႈၵႂႃႇၼႆ Room တေၼပ်ႉ 1, 2, 3, 4... ပၼ်ႁင်းၵွႆး ၼႆယဝ်ႉ။

---

### **3. Columns (သဝ်ၶေႃႈမူၼ်း)**

Variable ၵူႈတူဝ် ဢၼ်ယူႇၼႂ်း Data Class တေထုၵ်ႇပဵၼ် "သဝ်" (Column) ၼႂ်း Database ႁင်းၵွႆးၶႃႈယဝ်ႉ။ 
ၵူၺ်းၵႃႈ သင်ႁဝ်းၶႂ်ႈမႄးၸိုဝ်ႈသဝ်မၼ်း ႁႂ်ႈတႅတ်ႈတေႃးလိူဝ်ၵဝ်ႇၸိုင် ႁဝ်းၸၢင်ႈၸႂ်ႉ **`@ColumnInfo`** လႆႈယဝ်ႉ။

#### **တူဝ်ယၢင်ႇ Code (Expense.kt):**

```kotlin
@Entity(tableName = "expense_table")
data class Expense(
    @PrimaryKey(autoGenerate = true)
    val id: Int = 0,

    @ColumnInfo(name = "item_name")
    val name: String,

    @ColumnInfo(name = "amount_spent")
    val amount: Double,

    val date: Long // သင်ဢမ်ႇသႂ်ႇ ColumnInfo ၼႆၸိုင် မၼ်းတေၸႂ်ႉၸိုဝ်ႈ "date" ၵမ်းလဵဝ်ယဝ်ႉ
)

```

### **Advice for Students (ၶေႃႈသႅပ်ႇပၼ်လုၵ်ႈႁဵၼ်း):**

'သင်သၢင်ႈ Entity ၼႆ ႁႂ်ႈမီး Primary Key တႃႇသေႇၼႃၶႃႈ!' 
ဝူၼ်ႉတူၺ်းဝႃႈ သင်ႁဝ်းမီးလုၵ်ႈႁဵၼ်း 100 ၵေႃႉ ဢၼ်ၸိုဝ်ႈ 'ၸၢႆးမၢဝ်း' မိူၼ်ၵၼ်တင်းမူတ်းၼႆ
ႁဝ်းတေၸႅၵ်ႇၶဝ်ဢမ်ႇလႆႈယဝ်ႉၶႃႈ။ ၵူၺ်းၵႃႈ သင်ၶဝ်မီး 'မၢႆၾၢင်လုၵ်ႈႁဵၼ်း' (ID) ဢၼ်ဢမ်ႇမိူၼ်ၵၼ်ၼႆ
ႁဝ်းတေၸၢင်ႈႁႃၶဝ်ထူပ်းလႆႈငၢႆႈငၢႆႈ။ ၼၼ်ႉယဝ်ႉပဵၼ်ၼႃႈတီႈၶွင် Primary Key ၶႃႈ!
