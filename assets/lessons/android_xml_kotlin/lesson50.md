

## **Lesson 50: Introduction to Persistence**

### **1. What is Persistence? (ၵၢၼ်သိမ်းၶေႃႈမုၼ်း ႁႂ်ႈမၼ်ႈၵိုမ်း)**

ၼႂ်းၵၢၼ်တႅမ်ႈ App ၼႆႉ **Persistence** ပွင်ႇဝႃႈ ၵၢၼ်သိမ်းၶေႃႈမုၼ်းဝႆႉၼႂ်း Memory ယႃႇႁႂ်ႈမၼ်းႁၢႆ (Non-volatile storage) ၼႆယဝ်ႉ။

* **RAM:** မိူဝ်ႈႁဝ်းပိုတ်ႇ App မႃးၼၼ်ႉ ၶေႃႈမုၼ်းယူႇၼႂ်း RAM ယဝ်ႉ။ ပေႃးပိၵ်ႉ App ပႅတ်ႈၵေႃႈ ၶေႃႈမုၼ်းၼႂ်း RAM ၵေႃႈ ႁၢႆၵႂႃႇၵမ်းလဵဝ်ယဝ်ႉ။
* **Persistence (Database):** ၶေႃႈမူၼ်းတေထုၵ်ႇသိမ်းဝႆႉၼႂ်း Disk (Storage) ၶွင်ၾူၼ်းႁဝ်းယဝ်ႉ။ ဢမ်ႇဝႃႈတေပိၵ်ႉ App ႁိုဝ် ပိၵ်ႉၾူၼ်းၵေႃႈယဝ်ႉ ၶေႃႈမုၼ်းတေတိုၵ်ႉယူႇမိူၼ်ၵဝ်ႇယဝ်ႉ။

---

### **2. Why Room? (ပဵၼ်သင်လူဝ်ႇၸႂ်ႉ Room?)**

မိူဝ်ႈၵွၼ်ႇ Android ၸႂ်ႉ **SQLite** တႃႇသိမ်းၶေႃႈမူၼ်းယဝ်ႉ။ ၵူၺ်းၵႃႈ SQLite မၼ်းယၢပ်ႇ၊ လူဝ်ႇတႅမ်ႈ Code ၼမ်၊ လႄႈ ၽိတ်းငၢႆႈၼႃႇ။ 
ၵွပ်ႈၼႆလႄႈ Google ၸင်ႇသၢင်ႈ **Room Library** မႃးတႅၼ်းယဝ်ႉ။

**လွင်ႈလီၶွင် Room:**

* **Compile-time Verification:** သင်ၸဝ်ႈၵဝ်ႇတႅမ်ႈ Code ႁွင်ႉၶေႃႈမုၼ်းၽိတ်း (SQL query ၽိတ်း) ၼႆ Room တေလၢတ်ႈၼႄၸဝ်ႈၵဝ်ႇ ၵမ်းလဵဝ်မိူဝ်ႈတိုၵ်ႉ Compile ယူႇလႄႈ ဢမ်ႇပႂ်ႉႁႂ်ႈ App လူႉၸင်ႇႁူႉယဝ်ႉ။
* **Integration:** မၼ်းႁဵတ်းၵၢၼ်ၸွမ်း **LiveData** လႄႈ **Coroutines** လႆႈလီၼႃႇ (Native Support)။
* **Abstraction:** ႁဝ်းဢမ်ႇလူဝ်ႇတႅမ်ႈ Code SQLite ဢၼ်ယုင်ႈယၢင်ႉၼၼ်ႉထႅင်ႈယဝ်ႉ။

---

### **3. The Offline-First Experience (ၵၢၼ်ၸႂ်ႉ App လူၺ်ႈဢမ်ႇမီး Internet)**

ပၢႆးဝူၼ်ႉ Persistence ၼႆႉ လမ်ႇလွင်ႈတွၼ်ႈတႃႇႁဵတ်းႁႂ်ႈ App ႁဝ်း "ၶႅမ်ႉ" တေႉတေႉ။

* **Caching:** မိူဝ်ႈမီး Internet ၼၼ်ႉ ႁဝ်းဢဝ်ၶေႃႈမုၼ်းတီႈ API (Module 6) မႃးသိမ်းဝႆႉၼႂ်း Room။
* **Offline Mode:** မိူဝ်ႈၵူၼ်းၸႂ်ႉယူႇတီႈဢၼ်ဢမ်ႇမီး Internet ၼႆ App ႁဝ်းတိုၵ်ႉၸၢင်ႈပိုတ်ႇၼႄၶေႃႈမူၼ်း ဢၼ်သိမ်းဝႆႉၼႂ်း Room ၼၼ်ႉပၼ်ၶဝ်လႆႈယူႇယဝ်ႉ။

---

### **4. Understanding SQLite (ပိုၼ်ႉထၢၼ် Database)**

Room ၼႆႉ မၼ်းၵူၺ်းပဵၼ် "ၵူၼ်းႁေႃႇ" (Wrapper) ၶွင် SQLite ၵူၺ်း။ ၵွပ်ႈၼၼ် ႁဝ်းထုၵ်ႇပွင်ႇၸႂ်လွင်ႈ Database ဢိတ်းဢိတ်းယဝ်ႉ:

* **Table (ပႅၼ်ႈ):** တီႈသိမ်းၶေႃႈမုၼ်းပဵၼ်ၸုမ်း (မိူၼ်ၼင်ႇ သဵၼ်ႈၸိုဝ်ႈ "Expense" တႃႇသိမ်းလွင်ႈၸၢႆႇငိုၼ်း)။
* **Row (ထႅဝ်):** ၶေႃႈမုၼ်း 1 ဢၼ် (မိူၼ်ၼင်ႇ ၵၢၼ်ၸၢႆႇငိုၼ်းဝၼ်းမိူဝ်ႈၼႆႉ 10,000 ၵျၢပ်ႈ)။
* **Column (သဝ်):** ႁူဝ်ၶေႃႈၶေႃႈမူၼ်း (မိူၼ်ၼင်ႇ Id, Name, Amount, Date)။

---

### **Advice for Students**

'ဝူၼ်ႉတူၺ်းဝႃႈ Room မိူၼ်ၼင်ႇ "တိူၵ်ႈသိမ်းသဵၼ်ႈမၢႆ" (Filling Cabinet) ၼႂ်းႁူင်းၵၢၼ်ၼႆလႄႈ။' 
သင်ႁဝ်းဢမ်ႇမီးတိူၵ်ႈသိမ်းၼႆႉ၊ ပေႃးႁဝ်းပိၵ်ႉလုမ်းယဝ်ႉသေ ပွၵ်ႈမႃးၶိုၼ်းၼႆ ႁဝ်းတေလိုမ်းၵူႈပိူင်လႄႈ တေလႆႈ တေႇႁူဝ်ၶိုၼ်း ၵူႈဝၼ်းယဝ်ႉ။ 
ၵၢၼ်မီး Room တေႁဵတ်းႁႂ်ႈ App ႁဝ်းပဵၼ် App ဢၼ် "ႁူႉၵၢၼ်" လႄႈ တွပ်ႇလႆႈၵူႈယိူင်ႈ ဢၼ်ၵူၼ်းၸႂ်ႉယဝ်ႉ တႅမ်ႈဝႆႉၼၼ်ႉၶႃႈယဝ်ႉ!"
